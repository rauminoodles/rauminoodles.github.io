Code

Here is some of the code that I have developed for research. Not uploaded to the internet yet but stay tuned!

Lanczos

C++ code to diagonalize Hermitian matrices using the (Banded) Lanczos algorithm without re-orthogonalization.

       Features:

  • Lightweight template header-only library built on top of Eigen.
  • User-defined matrix-vector multiplication.

Noodle

(Noo)bie (D)iagonalization: (L)anczos and (E)xact. A simple and easy to implement header-only framework for constructing quantum many-body Hamiltonians. Diagonalizes the Hamiltonian matrix either exactly with Eigen or approximately with Lanczos.

       Features:

  • Lightweight template header-only library built on top of Eigen.
  • Can treat both fermions and bosons.
  • Compute: excitation spectrums and static & dynamic correlation functions (e.g. specific heat, Green's functions, dynamic structure factor, etc.).


       So far has treated:
  • Excitation spectrums and thermodynamic static correlation functions of spinless fermions and bosons on a sphere with a magnetic monopole at the center to study fractional quantum Hall physics.
  • Green's function and spin susceptibilities for: 1D & 2D Fermi-Hubbard model, 2D two-band Fermi-Hubbard model inter-orbital interactions, and 2D pi-flux Fermi-Hubbard model.
